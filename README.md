# Project 3D Metroidvania
This is a school project made by students from FutureGames Umeå.
Organized by Carl Leong and Tim Fagerdal 

Project 3D Metroidvania is a third person, 3D Metroidvania / platformer. Where you play as a person that has been tasked with trying to repair the timeline by the Guardian of Time because you stole a time-piece that grants you rudimentary control. 

Inspired by platformers from the PS2 era especially for example Jak and Daxter 1 and Sly 2. Having a focus on movement and handling, feeling good to use.
